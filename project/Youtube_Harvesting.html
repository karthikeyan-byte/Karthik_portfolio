YouTube Data Harvesting and Analysis Platform

This project is a robust platform for harvesting and analyzing YouTube data, including channel statistics, video insights, and audience engagement metrics. Built with a combination of Python libraries, MongoDB, SQLite, and Streamlit, the platform allows users to explore detailed analytics on YouTube channels and videos through an interactive web interface. To streamline deployment, Firebase provides user authentication, Docker enables containerization, and the application is hosted on Google Cloud Platform (GCP) for scalability and accessibility. Visit Application ➡️

1. Key Features

  • User Authentication: Secure login and signup pages powered by Firebase.
  • YouTube API Integration: Real-time data harvesting from YouTube channels and videos.
  • Data Storage and Conversion: JSON and CSV formats used for storage; data stored in SQLite for querying.
  • Interactive Data Analysis: Streamlit-based interactive dashboard for exploring video performance, likes, views, and more.
  • Containerized Deployment: Docker ensures consistency across environments.
  • Cloud Hosting: Deployed on Google Cloud for global accessibility and reliability.

2. Tech Stack

Technology Purpose
Python Backend processing and API interaction
Streamlit Frontend visualization
Firebase User authentication
SQLite & MongoDB Data storage for channel and video data
Docker Containerization for consistent deployment
Google Cloud Hosting for web accessibility and scalability

3. Project Architecture

  1. User Authentication: Firebase Authentication handles login and signup processes.
  2. YouTube Data Collection: Retrieves YouTube channel and video data using the YouTube Data API v3.
  3. Data Storage and Processing: Video data is stored in MongoDB and SQLite for easy querying.
  4. Data Visualization and Interaction: Streamlit dashboard offers interactive exploration of data.
  5. Containerization and Deployment: Docker packages the app for deployment on Google Cloud Platform (GCP).

4. Step-by-Step Workflow

  1. User logs in using Firebase Authentication.
  2. User inputs a YouTube channel name to retrieve channel and video data.
  3. Data is stored in MongoDB and SQLite for fast querying.
  4. Streamlit dashboard displays data visualizations for user interaction.
  5. The Docker container is deployed on GCP for global accessibility.

5. Data Analysis and User Queries

Users can analyze YouTube data based on various criteria, such as:

  • Top videos by view count
  • Channels with the most subscribers
  • Audience engagement metrics
  • Publishing patterns by year

6. Security and Scalability

  • Firebase Authentication: Provides secure, scalable user management.
  • Docker and GCP: Ensures consistent deployment with high availability on Google Cloud Platform.

7. Future Improvements

  • Advanced filtering options for date ranges and video categories.
  • Real-time data updates for continuously updated insights.
  • Expanded cloud integrations for deeper analytics with tools like BigQuery.

Conclusion

This YouTube Data Harvesting Platform is a scalable, data-driven solution for gaining insights into YouTube channel performance. With a secure, user-friendly interface, interactive data visualization, and real-time data fetching, it empowers users to explore YouTube analytics efficiently. The combination of Firebase for secure authentication, Docker for consistent deployment, and GCP for cloud hosting highlights a robust development approach for delivering scalable, production-ready applications.
